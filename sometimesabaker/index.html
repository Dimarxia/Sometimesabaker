🚀 Option A — GitHub Pages (recommended)

1
Create a repo named sometimesabaker, upload this folder.
2
Settings → Pages → Source: main branch, root. Save.
3
Live at yourusername.github.io/sometimesabaker in ~1 min.
iPhone: Safari → Share → "Add to Home Screen" = real app icon.

⚡ Option B — Netlify (drag & drop)

1
Go to netlify.com, drag your project folder onto the dashboard. Done.

🔑 API key security for public URLs

Personal / local: localStorage is fine.
Public URL: wrap the key in a Netlify Function and change API_ENDPOINT to /.netlify/functions/claude.
Netlify Functions free tier: 125k req/month — plenty for personal use.

📋 Adding new cards

1
Drop the generated .html file into recipes/.
2
Add one entry to recipes.json (title, category, tags, filename, swatch).
3
Commit / push (GitHub) or re-drag (Netlify) — grid updates automatically.
